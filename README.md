#### Please follow the below steps to build, start the application and run the tests

#### 1. build containers
docker-compose build

#### 2. start containers
docker-compose up

#### 3. run the app
after step 1 and 2 have been completed, navigate to http://localhost:4000 

#### 4. run BE tests
- navigate to server 
- install dependencies - npm install 
- run tests - npm run test

#### 5. running FE tests
- navigate to client 
- install dependencies - npm install 
- run tests - npm run test
